# 🔐 Cyber Security Portfolio – Future Interns (2026)

## 📌 Overview

This repository contains three practical cybersecurity projects completed as part of the **Future Interns Cyber Security Program**.

The projects demonstrate real-world security analysis skills across web applications, phishing detection, and API security.

---

## 📂 Projects Included

### 1️⃣ Vulnerability Assessment Report

* Conducted a read-only security assessment on a live website
* Identified security weaknesses such as missing headers and configuration issues
* Provided risk classification and remediation recommendations

---

### 2️⃣ Phishing Email Detection & Awareness

* Analyzed phishing email samples and headers
* Identified common phishing indicators and social engineering tactics
* Developed user awareness and prevention guidelines

---

### 3️⃣ API Security Risk Analysis

* Assessed a public API using Postman
* Identified risks such as lack of authentication and data exposure
* Provided practical API security recommendations
